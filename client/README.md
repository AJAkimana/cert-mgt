# Client

This is the web app for Sec CERTIFICATE. It lets users:

- Log in and manage templates
- Fill placeholders to generate certificates
- See generated files and download them

## Tech used

- Angular
- Tailwind CSS
- Angular Material

## Folder structure

- `src/app/core` services, guards, models
- `src/app/features` pages and UI
- `src/environments` API base URL

## Run locally

```bash
npm install
npm run start
```

Open `http://localhost:4200`.

## Build

```bash
npm run build
```
